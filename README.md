# Class Path Finder

A web application to help students find their class paths and rooms.

## Features

- **Class Schedule Parsing**: Upload PDF or Excel files to automatically extract class schedules.
- **Room Filtering**: Filter classes by day, block, and type.
- **Supabase Integration**: Data is stored and managed using Supabase.

## Setup

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Set up environment variables (see `.env.example`).
4.  Run the development server: `npm run dev`

## Tech Stack

- React
- Vite
- Supabase
